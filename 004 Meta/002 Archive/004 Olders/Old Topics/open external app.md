### How to open external app from obsidian
Just write `android-app://` after that write package name of selected app (APK name) usually begins with `com` after it write dot  `.` and apo name
**Example:**
- [Termux](android-app://com.termux)
- [deepseek](android-app://com.deepseek.chat)

***note***
There is here some apps like telegram have specific deeplink like YouTube
>in this case you can Read the docs and use the deeplink and you can use normal link (by Android-app://)

**Example:**
for YouTube we use [that's link (deeplink)](vnd.youtube://) or [that's link (app link)](android-app://com.google.android.youtube)
