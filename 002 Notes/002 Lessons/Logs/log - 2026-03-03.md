- the 4 laws or type of request is
    1.  **GET Request:** Used to **retrieve** data from the server. (e.g., getting a list of products, fetching a single user's profile).
    2.  **POST Request:** Used to **create** new data on the server. (e.g., submitting a new form, creating a new user account).
    3.  **PUT Request:** Used to **update** or **replace** existing data on the server. It typically replaces the *entire* resource with the new data provided. (e.g., updating all fields of a user's profile).
    4.  **DELETE Request:** Used to **remove** data from the server. (e.g., deleting a specific product, removing a user account).
